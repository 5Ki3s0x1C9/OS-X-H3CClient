H3CClientX

An OS-X-Compatible 802.1x Client

View project onGitHub

Introduction

H3CClientX is an H3C-specialized 802.1x protocol authentication client including a front-end monitor, based on previous work h3cclient b1 by asklux. This client has been verified working for students from Shandong University.
